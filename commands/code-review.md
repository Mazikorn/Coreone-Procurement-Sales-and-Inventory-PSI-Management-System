# /code-review

触发 code-reviewer 代理审查当前代码变更。

## 使用方式

1. 完成代码修改后，输入 `/code-review`
2. 或直接在对话中请求："帮我审查刚才的代码"

## 审查范围

代理会自动分析：
- 代码质量和可维护性
- 是否符合 COREONE 编码规范
- 类型安全和错误处理
- 潜在 bug 和性能问题

## 输出

- 问题列表（按 CRITICAL/HIGH/MEDIUM/LOW 分级）
- 具体修复建议（含代码示例）
- 正面反馈

## 注意事项

- 大型变更可分批审查
- 审查后根据建议修复，再请求二次审查
- 安全相关问题必须修复
