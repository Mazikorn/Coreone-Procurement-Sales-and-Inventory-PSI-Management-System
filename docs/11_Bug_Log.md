## P0 — Critical/High Bug

### BUG-001 BOM PUT 更新无事务包裹

| 字段 | 内容 |
|------|------|
| Bug ID | BUG-001 |
| 严重度 | Critical |
| 模块 | BOM 管理 |
| 状态 | 确认 |
| 来源 | QA Critical 确认、Plan 2 |
| 默认修复方 | mimo |

**复现步骤**
1. 编辑 BOM，修改物料明细
2. 系统先 DELETE 旧物料，再 INSERT 新物料
3. 若 INSERT 中间失败（如物料不存在），旧物料已被删除

**实际结果**：BOM 变成无物料状态
**期望结果**：DELETE 和 INSERT 在同一事务中，失败时全部回滚
**关联规则**：BR-BM-039（更新采用先删后插模式）
