## 2. 登录与导航

### 2.1 登录

1. 访问 `http://localhost:8080`
2. 输入用户名和密码
3. 点击「登录」或按 Enter
4. 登录成功后跳转到仪表盘

**注意事项**:
- 连续 5 次密码错误将锁定账户 15 分钟
- Token 有效期 8 小时，Refresh Token 有效期 7 天
- 支持「记住用户名」功能

### 2.2 界面导航

- **左侧菜单**: 根据角色动态显示可用功能
- **顶部导航**: 用户信息、退出登录
- **仪表盘**: 快速入口、统计概览、最近活动

## 3. 基础数据管理

### 3.1 物料分类 (`/categories`)

**功能**: 管理三级物料分类体系

**操作**:
- **查看分类树**: 页面左侧展示三级分类树，点击展开/折叠
- **新增分类**: 点击「新增」→ 选择父级分类 → 输入名称 → 保存
- **编辑分类**: 点击分类右侧「编辑」图标 → 修改名称 → 保存
- **删除分类**: 点击「删除」→ 确认（如有子分类，子分类自动提升到父级）
- **搜索**: 在搜索框输入关键词，实时过滤分类树

**数据统计**: 页面顶部显示总分类数、关联物料数、三级分类数

### 3.2 物料管理 (`/materials`)

**功能**: 管理所有物料的主数据

**操作**:
- **新增物料**: 点击「新增」→ 填写物料编码（自动按分类生成）、名称、规格、单位、单价、安全库存 → 保存
- **编辑物料**: 点击行末「编辑」→ 修改信息 → 保存
- **查看详情**: 点击物料名称 → 弹出详情面板（含批次信息、库存变动记录）
- **启用/停用**: 点击状态开关切换
- **批量操作**: 勾选多个物料 → 批量启用/停用/删除
- **筛选**: 按分类、供应商、状态筛选；支持关键词搜索

**业务规则**:
- 物料编码自动生成，格式: `MAT-{分类}-{序号}`
- 有库存的物料不能删除
- 停用的物料不能入库/出库

### 3.3 供应商管理 (`/suppliers`)

**功能**: 管理供应商信息

**操作**:
- **新增供应商**: 点击「新增」→ 填写名称、联系人、电话、地址 → 保存
- **编辑**: 修改供应商信息
- **评分**: 点击「自动评分」→ 系统根据采购金额(40%)、退货率(30%)、采购频次(30%)自动计算
- **状态管理**: 启用/停用供应商
- **批量操作**: 批量启用/停用/删除

**数据统计**: 总供应商数、合作中、已终止、本月新增

### 3.4 库位管理 (`/locations`)

**功能**: 管理仓库库位层级结构

**操作**:
- **查看库位树**: 左侧展示库位层级（区→架→位）
- **新增库位**: 点击「新增」→ 选择父级 → 输入名称、区、架、位、容量 → 保存
- **编辑**: 修改库位信息
- **删除**: 删除库位（需确认）
- **层级配置**: 自定义层级结构

**库位编码格式**: `LOC-{区}{架}{位}`，如 `LOC-A01` 表示 A区-01架-01位

### 3.5 检测服务 (`/projects`)

**功能**: 管理检测项目（HE/IHC/SS/MP/CYTO）

**操作**:
- **新增项目**: 点击「新增」→ 选择类型(he/ihc/ss/mp/cyto) → 填写名称、编码 → 关联 BOM → 保存
- **编辑**: 修改项目信息（支持 Tab 切换编辑）
- **复制**: 快速复制已有项目
- **导入**: 从 Excel 批量导入
- **批量启用/停用**: 勾选后批量操作

**数据统计**: 总项目数、已启用、未启用、未配置 BOM

## 4. 库存管理

### 4.1 库存列表 (`/inventory`)

**功能**: 查看所有物料的库存状态

**三个 Tab 页**:
- **在库**: 当前有库存的物料，显示库存数量、库位、批次信息
- **使用中**: 已出库但未耗尽的批次（消耗跟踪）
- **已耗尽**: 库存为 0 的物料

**操作**:
- **出库登记**: 选择物料 → 填写出库数量 → 选择出库类型 → 提交
- **批量出库**: 勾选多个物料 → 批量出库
- **批量报废**: 勾选多个物料 → 批量报废
- **查看详情**: 点击物料行 → 查看批次明细、库存变动记录

**库存状态标识**:
- 🟢 正常: 库存 > 安全库存
- 🟡 偏低: 库存 ≤ 安全库存
- 🔴 缺货: 库存 = 0
- ⏰ 临近过期: 批次过期日期 ≤ 30天
- ❌ 已过期: 批次已过期

## 5. 入库管理 (`/inbound`)

### 5.1 功能概述

管理所有入库操作，支持采购入库、退货入库、调拨入库等。

### 5.2 操作流程

#### 新增入库

1. 点击「新增入库」
2. 选择入库类型（采购入库/退货入库/其他）
3. 选择物料（下拉搜索）
4. 填写数量、单价、批次号
5. 选择供应商（采购入库必填）
6. 点击「保存」

#### 扫码入库

1. 点击「扫码入库」
2. 扫描物料二维码
3. 自动填充物料信息
4. 填写数量 → 保存

#### 批量导入

1. 点击「批量导入」
2. 下载 Excel 模板
3. 填写数据后上传
4. 系统校验并导入

#### 其他操作

- **编辑**: 修改入库记录（仅限未关联出库的记录）
- **删除**: 删除入库记录（自动回退库存）
- **查看详情**: 查看入库明细、关联批次
- **打印**: 打印入库单
- **导出**: 导出选中记录到 Excel

### 5.3 业务规则

- 入库自动创建批次记录
- 入库自动更新库存数量
- 入库自动关联采购订单（如有关联）
- 入库触发库存预警检查
- 已关联出库的入库记录不能删除

### 5.4 数据统计

页面顶部显示: 入库总数、总金额、待处理订单、供应商数量

## 6. 出库管理 (`/outbound`)

### 6.1 功能概述

管理所有出库操作，支持手动出库和 BOM 自动出库。

### 6.2 操作流程

#### 手动出库

1. 点击「新增出库」
2. 选择出库类型（项目出库/调拨出库/报废出库）
3. 选择关联项目（项目出库必填）
4. 添加物料行 → 选择物料 → 填写数量
5. 系统自动按 FEFO 原则分配批次（先到期先出）
6. 点击「保存」

#### BOM 出库（核心流程）

1. 点击「BOM 出库」
2. 选择检测项目
3. 系统自动加载 BOM 物料清单
4. 填写样本数量
5. 系统自动计算:
   - BOM 物料用量 × 样本数
   - 通用试剂分摊
   - 通用耗材分摊
   - 质控品分摊
   - ABC 滑动成本
   - 收费标准对比
6. 确认 → 保存

#### 其他操作

- **编辑**: 修改出库记录
- **删除/作废**: 作废出库记录（自动恢复库存，需填写作废原因）
- **查看详情**: 查看出库明细、ABC 成本分解
- **打印**: 打印出库单
- **批量导出**: 导出到 Excel（含 ABC 成本、费用、利润列）

### 6.3 业务规则

- 出库按 FEFO（先到期先出）原则自动分配批次
- 库存不足时返回 422 错误
- 出库自动创建消耗跟踪记录
- 出库触发 ABC 成本计算
- 出库触发库存预警检查

## 7. 盘点管理 (`/stocktaking`)

### 7.1 功能概述

管理库存盘点，支持创建盘点任务、录入实盘数量、差异调整。

### 7.2 操作流程

1. 点击「新建盘点」
2. 选择盘点范围（全部物料/指定分类/指定库位）
3. 系统生成盘点清单（物料 + 系统库存）
4. 逐一录入实盘数量
5. 系统自动计算差异
6. 确认盘点 → 自动调整库存

### 7.3 业务规则

- 盘点确认后自动调整库存（盘盈增加、盘亏减少）
- 盘点不允许产生负库存
- 已确认的盘点可以撤销（恢复调整前的库存）

### 7.4 数据统计

页面顶部: 进行中盘点数、已完成盘点数、待处理差异、准确率

## 8. 退库/报废/调拨

### 8.1 退库管理 (`/returns`)

**功能**: 管理物料退库（从使用端退回库存）

**操作**:
1. 点击「新增退库」
2. 选择物料 → 填写数量 → 选择退库原因
3. 退库原因: 质量问题 / 错发 / 过期 / 多余 / 其他
4. 保存 → 自动恢复库存

**业务规则**:
- 退库自动追溯最近出库记录，获取成本
- 退库恢复库存和批次数量
- 退库可以撤销（自动扣减库存）

### 8.2 报废管理 (`/scraps`)

**功能**: 管理物料报废

**操作**:
1. 点击「新增报废」
2. 选择物料 → 填写数量 → 选择报废原因
3. 报废原因: 过期 / 损坏 / 质量问题 / 淘汰 / 其他
4. 保存 → 自动扣减库存

**业务规则**:
- 报废扣减库存，不允许产生负库存
- 报废可以撤销（自动恢复库存）

### 8.3 调拨管理 (`/transfers`)

**功能**: 管理库位间物料调拨

**操作**:
1. 点击「新增调拨」
2. 选择物料 → 填写数量
3. 选择来源库位 → 选择目标库位
4. 保存 → 自动变更库位

**业务规则**:
- 调拨只变更库位，不改变总库存数量
- 来源库位必须与当前库存库位一致
- 调拨可以撤销

## 9. 供应商退货 (`/supplier-returns`)

### 9.1 功能概述

管理向供应商退货的完整生命周期。

### 9.2 状态流转

```
pending（待处理）→ shipped（已发货）→ received（供应商已收货）→ refunded（已退款）
     ↓
  cancelled（已取消）— 任何状态可取消
```

### 9.3 操作流程

1. 点击「新增退货」
2. 选择物料 → 选择批次 → 填写数量
3. 选择供应商 → 关联采购订单/入库单
4. 填写退货原因、退款金额、快递单号
5. 保存 → 自动扣减库存

6. **状态流转**:
   - 标记「已发货」→ 填写快递信息
   - 标记「供应商已收货」→ 确认供应商收货
   - 标记「已退款」→ 确认退款到账

### 9.4 业务规则

- 退货扣减库存和批次数量
- 仅 `pending` 状态可以删除
- 已退款的记录不能删除

## 10. BOM 管理 (`/bom`)

### 10.1 功能概述

管理检测项目的物料清单（Bill of Materials），是成本计算的核心。

### 10.2 操作流程

#### 新增 BOM

1. 点击「新增 BOM」
2. 填写 BOM 编码、名称、类型
3. 添加物料行:
   - 选择物料 → 填写每次用量 → 选择单位
   - 支持品牌池（group_name）: 同一用途多品牌可替代
4. 添加通用试剂: 选择通用试剂 → 填写分摊比例
5. 添加通用耗材: 选择耗材 → 填写分摊比例
6. 添加质控品: 选择质控品 → 填写频率
7. 添加设备模板: 选择设备 → 填写使用时长
8. 保存 → 自动计算标准成本

#### 其他操作

- **编辑**: 修改 BOM（自动递增版本号）
- **复制**: 快速复制已有 BOM
- **导入/导出**: 文件导入导出
- **成本预览**: 查看 BOM 成本分解（物料+人工+设备+间接）
- **查看详情**: Tab 切换查看各子项

### 10.3 数据统计

总 BOM 数、库存充足、库存偏低、库存不足

## 12. 成本管理

### 12.1 标准工时库 (`/labor-times`)

**功能**: 定义各检测步骤的标准工时

**操作**:
- **新增**: 填写步骤编码、名称、项目类型、标准分钟、费率/分钟、设备/人工类型
- **编辑/删除**: 修改或删除工时定义

**数据统计**: 总定义数、总分钟数、平均费率、设备步骤数

### 12.2 间接成本中心 (`/indirect-costs`)

**功能**: 管理间接成本（房租、水电、管理费等）

**操作**:
- **新增成本中心**: 填写名称、编码、成本类型、月度金额、分摊基数
- **成本分摊**: 选择月份 → 填写分摊金额 → 系统自动计算分摊率
- **编辑/删除**: 修改或删除

### 12.3 物料成本分析 (`/cost-analysis`)

**功能**: 6 个维度的成本分析

**Tab 页**:
1. **检测项目成本**: 按项目汇总出库成本
2. **项目分组成本**: 按 BOM 分组的成本分析
3. **物料消耗分析**: 按物料的消耗量和金额

## 18. 异常处理指南

### 18.1 常见错误码

| 错误码 | HTTP 状态 | 说明 | 处理方式 |
|--------|----------|------|---------|
| INVALID_PARAMETER | 400 | 请求参数错误 | 检查必填字段和格式 |
| NOT_FOUND | 404 | 资源不存在 | 检查 ID 是否正确 |
| UNAUTHORIZED | 401 | 未认证 | 重新登录 |
| FORBIDDEN | 403 | 无权限 | 联系管理员授权 |
| STOCK_INSUFFICIENT | 422 | 库存不足 | 检查库存数量 |
| ALREADY_HANDLED | 400 | 已处理 | 刷新页面查看最新状态 |
| INTERNAL_ERROR | 500 | 服务器错误 | 联系技术支持 |

### 18.2 业务规则限制

| 场景 | 限制 | 说明 |
|------|------|------|
| 删除物料 | 有库存不能删除 | 先清空库存再删除 |
| 删除分类 | 有子分类时子分类提升 | 自动提升到父级 |
| 出库 | 库存不足返回 422 | 检查库存数量 |
| 盘点 | 不允许负库存 | 实盘数不能小于已出库数 |
| 采购单取消 | 已完成不能取消 | 只能取消 pending/partial |
| 供应商退货删除 | 仅 pending 可删 | 先取消状态流转 |
| 设备删除 | 有使用记录不能删除 | 先删除使用记录 |
| BOM 删除 | 有关联出库不能删除 | 先处理关联数据 |

### 18.3 故障排查

**问题**: 登录失败
- 检查用户名密码是否正确
- 检查账户是否被锁定（连续 5 次错误锁定 15 分钟）
- 检查后端服务是否正常（`curl http://localhost:3001/api/health`）

**问题**: 页面空白
- 检查浏览器控制台是否有错误
- 检查 Token 是否过期（重新登录）
- 检查网络连接

**问题**: 操作无响应
- 检查后端日志（`docker logs coreone-backend-dev`）
- 检查数据库连接
- 检查是否有并发冲突
