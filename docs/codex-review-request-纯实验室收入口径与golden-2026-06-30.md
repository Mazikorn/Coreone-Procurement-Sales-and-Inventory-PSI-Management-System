- 依据：国标 81 码把"病理诊断费"(OUT) 与"病理标本处理费"(IN) 拆开（`病理类项目收费代码-YZ.xlsx`）。

### 主张 B · 老码按比例拆（方案2）
- 老捆绑码（2703"检查与诊断"、TCT、冰冻切片）= 制片(IN)+诊断(OUT)，按 `制片份额 = 36×蜡块 / (36×蜡块 + 105)` 逐病例拆。费率：组织 ¥36/蜡块、细胞 ¥75/玻片、诊断 ¥105/病例（国标）。
- **请验**：拆分公式对吗？把国标比例套到和睦家溢价单价（¥165 vs 国标 ¥141）合理吗？"一病例一诊断费"（多标本共用 105）对吗？

## 3. 最该挑的（我自己担心的点）

4. **G2 工时/间接是国外锚**（26:40:34），套国内属弱。
5. **成本按院拆做不到**——毛利 ~54% 是"真收入 − 估算成本"，不是全实测。
