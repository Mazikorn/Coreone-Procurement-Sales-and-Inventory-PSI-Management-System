### 💰 里程碑 2：成本核算体系建立（5月21-29日）

**业务背景**：原有系统只能核算直接材料成本（占总成本35-50%），无法满足单切片全成本核算需求，影响定价策略和成本控制。

**行业调研结论**：
- 全球主流病理系统（Epic/Sunquest/SPD/国产LIS）均无单切片全成本核算功能
- 正确方案：扩展BOM + ABC作业成本法（物料+人工+设备+间接）
- 符合《上海市医院成本管理暂行办法》要求

**成本核算架构**：
```
单切片全成本 = 材料成本 + 人工成本 + 设备成本 + 质控成本 + 间接成本
     │              │          │          │          │
     │              │          │          │          └─ 房租/水电/管理费
     │              │          │          └──────────── 质控品消耗
     │              │          └─────────────────────── 设备折旧（工作量法）
     │              └────────────────────────────────── 标准工时 × 费率
     └────────────────────────────────────────────────── 物料消耗（FEFO）
```

**实现成果**：
| 模块 | 功能 | 状态 | 业务价值 |
|------|------|------|----------|
| BOM扩展 | 通用试剂/耗材/质控品/设备模板配置 | ✅ 完成 | 覆盖完整检测流程物料 |
| 品牌池替代 | 多品牌自动分配（FEFO） | ✅ 完成 | 降低采购成本，保障供应连续性 |
| Panel分组 | 检测项目分组成本统计 | ✅ 完成 | 精细化成本分析 |
| 设备管理 | 设备档案+折旧计算（直线法/工作量法） | ✅ 完成 | 设备成本可追溯 |
| 标准工时库 | 10个标准步骤+费率配置 | ✅ 完成 | 人工成本可量化 |
| 间接成本中心 | 房租/水电/管理费分摊 | ✅ 完成 | 间接成本可分摊 |
| 全成本报表 | 5项成本分项展示 | ✅ 完成 | 成本结构透明化 |

**业务价值**：
- 实现国内首个单切片全成本核算系统
- 成本核算覆盖率从35%提升至95%
- 支持定价策略优化和成本控制决策
