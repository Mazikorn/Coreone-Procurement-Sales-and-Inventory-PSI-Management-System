## 0. 概述

**系统名称**: COREONE — 病理免疫组化特染进销存与单张切片成本控制系统
**系统定位**: B 端实验室耗材管理平台
**目标用户**: 病理免疫组化特染实验室
**核心价值**: 耗材全链路追踪 + 单张切片精确成本核算

**技术基线**: React 18 + TypeScript + Vite 前端，Node.js + Express + TypeScript + SQLite 后端

## 2. MOD-01 认证与登录

### 需求列表

| 需求编号 | 需求名称 | 用户故事 | 验收标准 |
|----------|----------|----------|----------|
| REQ-01-001 | 用户名密码登录 | 作为用户，我可以通过用户名和密码登录系统 | 输入正确凭据后获得 JWT token，跳转到仪表盘 |
| REQ-01-002 | Token 刷新 | 作为已登录用户，我的 token 可以自动刷新 | Access Token 过期后通过 Refresh Token 自动获取新 token |
| REQ-01-003 | 用户登出 | 作为已登录用户，我可以退出登录 | 点击登出后清除本地 token，跳转到登录页 |

### API 端点

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/api/v1/auth/login` | 登录（公开） |
| POST | `/api/v1/auth/refresh` | 刷新 token |
| POST | `/api/v1/auth/logout` | 登出（公开） |
