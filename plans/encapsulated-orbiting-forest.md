## 第一阶段：后端API测试补充（真实工作流覆盖）

### 1.1 入库管理真实路径测试（`tests/inbound-real-flow.test.ts`）

**场景A：采购入库完整流程（正常路径）**
```
创建采购订单 → 采购入库 → 库存确认 → 批次验证 → 预警触发
```
- 步骤1：创建采购订单（HPV检测试剂盒，数量100，预期到货日期）
- 步骤2：采购入库（关联PO，录入批次号/有效期，数量100）
- 步骤3：验证库存增加100
- 步骤4：验证批次创建（批次号、有效期、入库价）
- 步骤5：验证库存预警（低库存/即将过期）

**场景B：入库取消与恢复（异常路径）**
```
采购入库 → 取消入库 → 库存回退 → 恢复入库 → 库存恢复
```
- 验证取消后库存回退
- 验证恢复后库存恢复
- 验证取消后批次状态变化

**场景C：入库修改与成本回溯（边界路径）**
```
采购入库@100元 → 出库消耗50 → 修改入库价@120元 → 验证成本变化
```
- 验证修改入库单数量/价格后的库存和成本联动

**场景D：直接入库（无PO路径）**
```
直接入库 → 库存确认 → 无PO关联验证
```

### 1.2 出库管理真实路径测试（`tests/outbound-real-flow.test.ts`）

**场景A：项目领用BOM出库（正常路径）**
```
选择检测项目（乳腺癌分子分型）→ BOM一键出库 → 批次FEFO分配 → 成本归集 → 库存扣减
```
- 步骤1：获取乳腺癌分子分型项目（IHC-BR-01）及其BOM
- 步骤2：确认物料库存充足（ER抗体、PR抗体、HER2抗体、Ki-67抗体）
- 步骤3：BOM出库10个样本
- 步骤4：验证按FEFO分配批次
- 步骤5：验证库存扣减正确
- 步骤6：验证项目成本归集正确
- 步骤7：验证全成本明细（材料+人工+设备+质控+间接）

**场景B：品牌池替代出库（替代路径）**
```
主品牌(Dako)库存不足 → 自动替代(迈新) → 成本按替代品牌计算
```
- 步骤1：创建物料Ki-67(Dako)入库5个
- 步骤2：创建物料Ki-67(迈新)入库10个
- 步骤3：BOM出库8个（Dako不足，自动从迈新分配3个）
- 步骤4：验证成本计算（5*Dako价 + 3*迈新价）

**场景C：库存不足出库（异常路径）**
```
库存不足 → 出库失败 → 错误提示
```
- 验证返回正确的错误码和错误消息

**场景D：出库修改与删除（回溯路径）**
```
BOM出库 → 修改数量 → 库存回退重分配 → 删除出库 → 完全恢复
```
- 验证修改后库存和成本重新计算
- 验证删除后库存完全恢复

### 1.3 设备管理真实路径测试（`tests/equipment-real-flow.test.ts`）

**场景A：设备创建与折旧计算（正常路径）**
```
创建设备（免疫组化染色机，15万，5年）→ 直线法折旧 → 验证月折旧额
```
- 验证直线法：月折旧 = (150000 - 残值) / (5*12)

**场景B：工作量法折旧（替代路径）**
```
创建设备（切片机，10万，总容量10万张）→ 工作量法 → 使用1000张 → 验证折旧
```
- 验证工作量法：单张折旧 = (100000 - 残值) / 100000

**场景C：设备使用登记（联动路径）**
```
设备创建 → 项目出库 → 自动登记使用 → 折旧成本分摊
```
- 验证设备使用记录创建
- 验证折旧成本正确分摊到项目
