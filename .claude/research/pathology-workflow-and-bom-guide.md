## 一、病理科核心工作流

病理科日常工作围绕**"标本接收→制片→染色→诊断→报告"**五大环节展开，各环节对应不同的耗材消耗和检测项目。

```
┌──────────┐   ┌──────────┐   ┌──────────┐   ┌──────────┐   ┌──────────┐
│ 标本接收 │ → │ 制片处理 │ → │ 染色处理 │ → │ 阅片诊断 │ → │ 出具报告 │
└────┬─────┘   └────┬─────┘   └────┬─────┘   └────┬─────┘   └────┬─────┘
     │              │              │              │              │
     ▼              ▼              ▼              ▼              ▼
  标本袋/瓶    固定液/脱水液   HE/IHC/SS      显微镜/油镜     标签纸/报告纸
  标签纸       包埋盒/石蜡    抗体/显色液    图文报告系统    存储归档
  登记本       刀片/玻片      缓冲液/封片剂  诊断试剂
```

### 1.1 制片流程

| 步骤 | 操作内容 | 对应耗材 |
|------|---------|---------|
| 固定 | 10%中性福尔马林固定（6-48h） | 福尔马林固定液、标本袋/瓶 |
| 脱水 | 梯度乙醇脱水（75%→95%→100%） | 乙醇（各浓度）、二甲苯/脱蜡液 |
| 透明 | 二甲苯透明 | 二甲苯 |
| 浸蜡 | 石蜡浸透 | 石蜡（56-58℃） |
| 包埋 | 组织包埋成蜡块 | 包埋盒、包埋底模 |
| 切片 | 石蜡切片（3-5μm）或冰冻切片 | 病理刀片、载玻片（防脱） |
| 烤片 | 60℃烤片30min | 烤箱/烤片机 |

### 1.2 染色分类

| 染色类型 | 代号 | 主要用途 | 检测时长 |
|---------|------|---------|---------|
| HE染色 | he | 常规组织学诊断，最基础染色 | 30-60min |
| 免疫组化 | ihc | 肿瘤分型、来源鉴别、靶向标志物 | 2-4h（全自动） |
| 特殊染色 | ss | 特定组织成分显示（胶原、黏液、真菌等） | 30-120min |
| 分子病理 | mp | 基因突变、融合、扩增检测 | 1-3天 |
| 细胞学 | cyto | 脱落细胞/穿刺细胞学诊断 | 1-2h |

## 二、物料分类体系（三级分类）

基于病理科工作逻辑，物料分类采用**三级树形结构**，一级按功能域划分，二级按技术平台划分，三级按具体品类划分。

### 2.1 一级分类

| 编码前缀 | 分类名称 | 说明 |
|---------|---------|------|
| R- | 试剂类 | 所有化学试剂、生物试剂 |
| C- | 耗材类 | 一次性实验耗材、器具 |
| Q- | 质控品类 | 阳性质控、阴性质控 |
| D- | 设备配件类 | 设备耗材、维护配件 |

## 四、BOM清单结构

### 4.1 BOM类型定义

| BOM类型 | 说明 | 适用项目类型 |
|--------|------|------------|
| standard | 标准BOM | 所有项目类型 |

### 4.2 BOM物料构成（五层结构）

```
BOM (物料清单)
├── 特异性试剂 (bom_items)
│   └── 按项目特异性配置的抗体/试剂
├── 通用试剂配额 (bom_general_reagents)
│   └── DAB、PBS、抗原修复液、缓冲液等共享试剂
├── 通用耗材配额 (bom_general_consumables)
│   └── 载玻片、吸头、手套等共享耗材
├── 质控品配额 (bom_quality_controls)
│   └── 阳性质控、阴性质控（按批次覆盖样本数分摊）
└── 设备模板 (bom_equipment_templates)
    └── 切片机、染色仪、封片机等使用时长配置
```

## 五、成本核算逻辑

### 5.1 直接材料成本

```
直接材料成本 = Σ(特异性试剂用量 × 批次加权单价)
             + Σ(通用试剂用量 × 批次加权单价)
             + Σ(通用耗材用量 × 批次加权单价)
             + Σ(质控品用量 × 批次加权单价 / 覆盖样本数)
```

### 5.2 直接人工成本

```
直接人工成本 = Σ(各环节标准工时 × 工时费率) × 样本数
```

标准工时库预置环节：
- 标本接收：3min
- 切片：12min
- 抗原修复：25min（IHC/SS）
- 免疫组化染色：120min
- 特殊染色：90min
- 封片：5min
- 阅片诊断：20min
- HE染色：45min

### 5.3 设备折旧成本

```
直线法：年折旧额 = (采购价 - 残值) / 折旧年限
工作量法：单张折旧 = (采购价 - 残值) / 总工作量容量
```

### 5.4 间接费用分摊

```
间接费用分摊 = 当月间接费用总额 / 当月总样本数 × 该项目样本数
```

间接成本中心：
- 房租物业
- 水电能源
- 管理人员薪酬
- 设备维护费
- 信息系统费用

## 七、使用说明

### 7.1 本文档与COREONE系统的对应关系

| 文档章节 | 对应系统模块 | 对应数据库表 |
|---------|------------|------------|
| 二、物料分类体系 | 物料分类管理 | material_categories |
| 三、检测项目体系 | 检测项目管理 | projects |
| 四、BOM清单结构 | BOM管理 | boms, bom_items, bom_general_reagents, bom_general_consumables, bom_quality_controls, bom_equipment_templates |
| 五、成本核算逻辑 | 成本分析报表 | project_cost_details, standard_labor_times, equipment, indirect_cost_centers |

### 7.2 配置步骤

1. **第一步**: 在系统中录入物料分类（三级分类树）
2. **第二步**: 录入物料主数据（关联到三级分类）
3. **第三步**: 录入检测项目（关联项目类型）
4. **第四步**: 为每个检测项目配置BOM（五层结构）
5. **第五步**: 配置设备档案和标准工时库
6. **第六步**: 验证成本核算逻辑

### 7.3 注意事项

- **抗体替代**: 同一抗体不同品牌（如Dako、迈新、中杉金桥）可在BOM中设置为品牌池（替代物料）
- **通用试剂分摊**: 通用试剂（如PBS、DAB）按实际出库批次加权单价计算
- **质控品分摊**: 质控品成本按"每批次覆盖样本数"分摊到单个样本
- **效期管理**: 抗体、DAB等需2-8℃避光保存，系统需设置有效期预警
- **Panel概念**: 同一Panel内的抗体（如MMR四联检）建议设为同一分组
