---

## 一、产品定位

**用户需求**：病理实验室需要核算"单切片全成本"，包括材料、人工、设备、间接费用，以便进行成本控制和盈利分析。

**产品功能**：提供一套完整的成本管理工具，让用户能够：
1. 管理设备资产和折旧
2. 定义标准工时和费率
3. 录入和分摊间接费用
4. 配置BOM的成本构成
5. 查看多维度成本报表

**实现逻辑**：底层采用ABC作业成本法进行成本计算，但用户无需了解ABC的具体概念。

## 三、功能模块详细定义

### 3.1 设备管理

#### 用户故事

| ID | 用户故事 | 验收标准 |
|----|---------|---------|
| EM-01 | 作为设备管理员，我希望录入设备基本信息，以便建立设备档案 | 支持录入设备名称、型号、购置价格、购置日期、折旧年限等 |
| EM-02 | 作为设备管理员，我希望记录设备使用情况，以便计算设备成本 | 支持关联项目、记录使用时长、自动计算折旧成本 |
| EM-03 | 作为财务人员，我希望查看设备折旧统计，以便了解设备成本分布 | 按设备/项目/时间维度展示折旧统计 |

#### 功能清单

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 设备列表 | 展示所有设备，支持筛选、搜索、分页 | P0 |
| 设备详情 | 查看设备完整信息和使用历史 | P0 |
| 新增设备 | 录入设备基本信息和折旧规则 | P0 |
| 编辑设备 | 修改设备信息 | P0 |
| 删除设备 | 删除设备（需检查是否有关联记录） | P1 |
| 使用登记 | 记录设备使用情况（关联项目、使用时长） | P0 |
| 折旧统计 | 按设备/项目/时间维度统计折旧成本 | P0 |

#### 数据模型（用户视角）

```
设备档案
├── 基本信息：名称、型号、制造商
├── 财务信息：购置价格、购置日期、残值
├── 折旧规则：折旧年限、折旧方法（直线法/工作量法）
└── 状态：在用/闲置/报废
```

### 3.4 BOM成本配置

#### 用户故事

| ID | 用户故事 | 验收标准 |
|----|---------|---------|
| BC-01 | 作为BOM管理员，我希望为BOM配置通用试剂配额，以便计算试剂成本 | 支持添加DAB、PBS等通用试剂的标准用量 |
| BC-02 | 作为BOM管理员，我希望为BOM配置通用耗材配额，以便计算耗材成本 | 支持添加载玻片、吸头等通用耗材的标准用量 |
| BC-03 | 作为BOM管理员，我希望为BOM配置质控品配额，以便计算质控成本 | 支持添加阳性质控、阴性质控的标准用量 |
| BC-04 | 作为BOM管理员，我希望为BOM配置设备使用模板，以便计算设备成本 | 支持添加所需设备及使用时长 |

#### 功能清单

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 通用试剂配额 | 为BOM添加通用试剂（DAB、PBS等）的标准用量 | P0 |
| 通用耗材配额 | 为BOM添加通用耗材（载玻片、吸头等）的标准用量 | P0 |
| 质控品配额 | 为BOM添加质控品的标准用量和覆盖样本数 | P0 |
| 设备使用模板 | 为BOM添加所需设备及使用时长 | P0 |
| 成本预览 | 预览BOM的全成本构成（材料+人工+设备+间接） | P1 |

#### 数据模型（用户视角）

```
BOM成本配置
├── 通用试剂配额
│   ├── 试剂：DAB、PBS、缓冲液等
│   ├── 用量：每样本/每批次用量
│   └── 单位：ml、滴、次等
├── 通用耗材配额
│   ├── 耗材：载玻片、吸头、手套等
│   └── 用量：每样本用量
├── 质控品配额
│   ├── 质控品：阳性质控、阴性质控
│   ├── 用量：每批次用量
│   └── 覆盖样本数：每批次质控覆盖的样本数
└── 设备使用模板
    ├── 设备：切片机、染色机等
    └── 使用时长：分钟
```

### 3.5 成本报表

#### 用户故事

| ID | 用户故事 | 验收标准 |
|----|---------|---------|
| CR-01 | 作为财务人员，我希望查看全成本报表，以便了解各项目的真实成本 | 展示材料+人工+设备+间接的完整成本构成 |
| CR-02 | 作为财务人员，我希望查看成本差异分析，以便发现成本异常 | 对比标准成本与实际成本，分析价格差异和用量差异 |
| CR-03 | 作为管理者，我希望查看成本结构分析，以便了解成本构成 | 展示各成本要素的占比（饼图/柱状图） |
| CR-04 | 作为管理者，我希望查看盈利性分析，以便了解项目盈利情况 | 对比成本与收费，计算利润率 |

#### 功能清单

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 全成本报表 | 按项目/样本维度展示全成本构成 | P0 |
| 成本差异分析 | 对比标准成本与实际成本，分析差异原因 | P0 |
| 成本结构分析 | 展示各成本要素的占比（饼图/柱状图） | P0 |
| 盈利性分析 | 对比成本与收费，计算利润率 | P0 |
| 成本趋势 | 按时间维度展示成本变化趋势 | P1 |
| 成本导出 | 导出成本报表为Excel/PDF | P1 |

#### 数据模型（用户视角）

```
全成本报表
├── 项目信息：项目名称、项目类型、样本数
├── 成本构成
│   ├── 材料成本：特异性试剂 + 通用试剂 + 通用耗材 + 质控品
│   ├── 人工成本：各环节标准工时 × 费率
│   ├── 设备成本：设备使用时长 × 折旧费率
│   └── 间接成本：按分摊规则计算
├── 合计：总成本、单位成本（单切片成本）
└── 对比：标准成本、差异金额、差异率
```

## 四、功能与实现逻辑映射

| 产品功能 | 实现逻辑（隐藏） | 说明 |
|---------|----------------|------|
| 设备管理 | 作业中心（设备相关） | 用户管理设备，系统自动创建设备作业中心 |
| 标准工时 | 成本动因（人工相关） | 用户配置工时，系统自动创建人工成本动因 |
| 间接成本中心 | 成本池（间接费用） | 用户管理成本中心，系统自动归集成本池 |
| BOM成本配置 | BOM作业关联 | 用户配置BOM成本，系统自动创建作业关联 |
| 成本报表 | 成本计算+分配 | 用户查看报表，系统自动计算作业成本 |

## 六、待讨论问题

1. **设备管理的粒度**：是否需要管理到每台设备，还是按设备类型管理？
2. **标准工时的来源**：是用户手动配置，还是从历史数据自动学习？
3. **间接成本的分摊周期**：按月分摊，还是按季度/年度？
4. **BOM成本配置的位置**：在BOM管理页面内，还是独立的成本配置页面？
5. **成本报表的维度**：需要哪些维度的下钻（项目→样本→物料）？
