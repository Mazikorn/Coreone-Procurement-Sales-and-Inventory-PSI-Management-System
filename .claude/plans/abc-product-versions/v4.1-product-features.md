## 一、5why验证发现的问题

### 验证结果总览

| 层级 | 需求 | v4.0满足程度 | 主要缺失 |
|------|------|-------------|----------|
| 第一层 | 单切片全成本核算 | ✅ 基本满足 | 通用配额、权重系数定义 |
| 第二层 | 成本控制 | ⚠️ 部分满足 | 趋势分析、预警功能 |

1. **成本分析维度不足**：缺少趋势分析、对比分析、预警功能

3. **决策支持工具缺失**：缺少定价模拟、供应商对比、人员产能分析等工具
4. **运营效率指标缺失**：缺少切片产出率、设备利用率等关键指标

## 二、v4.1 改进方案

### P0 改进（必须实现）

#### 1. 补充通用配额和权重系数定义

**通用配额定义**：
```
通用配额 = 通用试剂配额 + 通用耗材配额 + 质控品配额

其中：
- 通用试剂配额：DAB、PBS、缓冲液等共享试剂的成本
- 通用耗材配额：载玻片、吸头、手套等共享耗材的成本
- 质控品配额：阳性质控、阴性质控的成本（按覆盖样本数分摊）
```

**权重系数定义**：
```
权重系数 = 该步骤标准耗时 / 所有步骤标准耗时之和

示例：
- HE染色步骤标准耗时：45分钟
- 所有步骤标准耗时之和：3+12+45+5+20 = 85分钟
- HE染色权重系数 = 45 / 85 = 0.529

设备成本 = 设备类型年折旧额 / 年切片数量 × 权重系数
```

```
盈利分析报表
├── 项目盈利排名（表格）
│   ├── 项目名称
│   ├── 切片数量
│   ├── 总成本

│   ├── 利润额
│   └── 利润率

│   ├── 中盈利项目（利润率10%-30%）

**功能描述**：
- 展示成本随时间的变化趋势

**页面设计**：
```
成本趋势分析
├── 趋势图表（折线图）
│   ├── X轴：时间（月/季度/年）
│   ├── Y轴：成本金额
│   └── 多条线：总成本、材料成本、人工成本、设备成本、间接成本

### P1 改进（重要，可选实现）

#### 4. 增加成本预警功能

**功能描述**：
- 设置成本阈值
- 超限时自动预警
- 支持多种预警规则

**预警规则**：
```
成本预警规则
├── 成本超支预警
│   ├── 条件：实际成本 > 标准成本 × (1 + 阈值)
│   └── 阈值：可配置，默认10%
├── 利润率预警
│   ├── 条件：利润率 < 最低利润率
│   └── 阈值：可配置，默认0%
├── 成本突增预警
│   ├── 条件：本月成本 > 上月成本 × (1 + 阈值)
│   └── 阈值：可配置，默认20%
└── 预警通知
    ├── 系统内通知
    └── 邮件通知（可选）
```

#### 5. 增加项目盈利排名

**功能描述**：
- 按盈利性对项目排序
- 快速识别高盈利/亏损项目
- 支持筛选和导出

**排名维度**：
```
项目盈利排名
├── 按利润率排序（高→低）
├── 按利润额排序（高→低）
├── 按切片数量排序（高→低）
└── 筛选条件
    ├── 时间范围
    ├── 项目类型
    └── 盈利区间（高盈利/中盈利/低盈利/亏损）
```

#### 6. 增加运营效率指标

**功能描述**：
- 展示关键运营效率指标
- 识别效率低下的环节
- 支持趋势分析

**指标定义**：
```
运营效率指标
├── 切片产出率 = 实际切片数 / 标准切片数
├── 设备利用率 = 设备实际使用时长 / 设备可用时长
├── 人工效率 = 标准工时 / 实际工时
├── 材料损耗率 = 实际材料消耗 / 标准材料消耗
└── 成本偏差率 = (实际成本 - 标准成本) / 标准成本
```

## 四、v4.1 成本计算公式（完善版）

```
单切片全成本 = 直接材料成本 + 人工成本 + 设备成本 + 间接成本

其中：
1. 直接材料成本 = Σ(组件用量 × 单价 × (1 + 损耗率)) + 通用配额
   - 通用配额 = 通用试剂配额 + 通用耗材配额 + 质控品配额

2. 人工成本 = Σ(标准工时 × 费率)

3. 设备成本 = Σ(设备类型年折旧额 / 年切片数量 × 权重系数)
   - 权重系数 = 该步骤标准耗时 / 所有步骤标准耗时之和

4. 间接成本 = Σ(成本中心月度费用 / 月度切片数量)
```

### 数据来源

| 成本项 | 数据来源 | 用户录入 |
|--------|---------|---------|
| 直接材料成本 | 物料单价（物料表）+ BOM用量 | BOM配置时录入用量 |
| 通用配额 | BOM配置（通用试剂/耗材/质控品配额） | BOM配置时录入 |
| 人工成本 | 标准工时（standard_labor_times表）+ 费率 | 配置时录入 |
| 设备成本 | 设备台账（equipment表）+ 切片统计 | 设备台账录入一次 |
| 间接成本 | 成本中心（indirect_cost_centers表）+ 月度费用 | 每月录入费用 |
