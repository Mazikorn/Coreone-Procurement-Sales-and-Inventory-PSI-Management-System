## 一、功能流程（5个模块）

### 1. 设备管理模块

#### 用户操作流程
```
管理员/仓库管理员 → 进入设备管理页面
  ├── 查看设备列表（分页、筛选、搜索）
  ├── 新增设备 → 填写设备台账信息 → 保存
  ├── 编辑设备 → 修改信息 → 保存
  ├── 查看设备详情 → 查看使用记录、折旧统计
  ├── 管理设备类型 → 新增/编辑/删除设备类型
  └── 查看设备折旧统计 → 按类型/项目/时间维度查看
```

#### 数据流转
```
equipment 表 → equipment_usage 表 → bom_equipment_templates 表 → outbound_abc_details 表
```

---

### 2. 标准工时管理模块

#### 用户操作流程
```
管理员/财务 → 进入标准工时管理页面
  ├── 查看作业环节列表（8个预设步骤）
  ├── 编辑标准工时 → 选择项目类型 → 输入标准耗时和费率
  ├── 查看参考值 → 显示来源标签 → 一键采纳
  └── 批量设置费率
```

#### 数据流转
```
standard_labor_times 表 → abc_bom_activity_links 表 → cost-calculator.ts → outbound_abc_details 表
```

---

### 3. 间接成本中心模块

#### 用户操作流程
```
财务/admin → 进入间接成本中心页面
  ├── 成本中心管理（CRUD）
  ├── 月度费用录入（选择成本中心+月份 → 录入金额和基数）
  ├── 季度调整（自动计算+手动触发 → 审核确认）
  └── 分摊记录查看
```

#### 数据流转
```
indirect_cost_centers 表 → indirect_cost_allocations 表 → cost-calculator.ts → outbound_abc_details 表
```

---

### 4. BOM 成本配置模块

#### 用户操作流程
```
管理员 → 进入 BOM 管理页面
  ├── Tab 1: 基本信息
  ├── Tab 2: 组件列表（物料+用量+损耗率）
  ├── Tab 3: 通用配额（试剂/耗材/质控品）
  ├── Tab 4: 设备模板（设备类型+使用时长）
  └── Tab 5: 成本预览（汇总+饼图+刷新按钮）
```

#### 数据流转
```
bom_components 表 → bom_general_quotas 表 → bom_equipment_templates 表 → cost-calculator.ts → 成本预览缓存
```

---

### 5. 成本报表模块

#### 用户操作流程
```
所有角色 → 项目级汇总（成本结构+盈利性分析）
  ↓ 点击项目（需权限）
财务/主管/admin → 样本级下钻（成本明细+差异分析）
  ↓ 点击样本（需权限）
财务/采购/admin → 物料级下钻（物料消耗+偏差分析）
  ↓
财务/admin → 导出（Excel/PDF）
```

#### 数据流转
```
outbound_records 表 → outbound_abc_details 表 → outbound_material_details 表 → 报表页面 → 导出文件
```

### 问题2：标准工时配置的粒度是什么？
