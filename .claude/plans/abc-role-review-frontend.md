## 发现的问题

### Critical（3 个）

1. **现有 ABC 页面未使用项目 API 层** - 全部使用原生 fetch()，绕过 request 模块，401 自动跳转等拦截器失效
2. **ProfitabilityAnalysis 调用的 API 端点不存在** - 该页面目前是死代码
3. **TypeScript 类型定义缺失** - ABC 数据结构无类型约束

### High（4 个）

4. **成本瀑布图组件需要从零构建** - Recharts 无原生瀑布图组件
5. **出库成本预览与出库表单的集成方案不清晰** - 需要新增 BOM 出库模式
6. **BOM 详情页缺少收费标准配置入口**
7. **盈利性分析页面不支持三维度切换**

### Medium（4 个）

8. **三个 ABC CRUD 页面存在大量重复代码** - 约 60% 是重复模式
9. **成本池管理的动因费率汇总计算有误** - 应使用加权平均
10. **成本趋势图的时间范围硬编码**
11. **出库详情弹窗缺少 ABC 成本信息**

### Low（2 个）

12. **导出功能是假实现**
