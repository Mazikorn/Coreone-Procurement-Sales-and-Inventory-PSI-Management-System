## 七、修复详情

### 7.1 Docker 容器重建

**问题**: Docker 容器运行旧版代码，导致 4 个 API 返回 404
**修复**: `docker compose -f docker-compose.dev.yml up -d --build`

### 7.2 浮点数精度修复

**文件**: `前端代码/src/pages/outbound/components/OutboundTable.tsx:165`
**修复**:
```tsx
// Before
{row.items?.reduce((sum, i) => sum + i.quantity, 0) || 0}

// After
{Number((row.items?.reduce((sum, i) => sum + i.quantity, 0) || 0).toFixed(2))}
```

### 7.3 错误提示优化

**文件**: `前端代码/src/pages/outbound/Outbound.tsx:264`
**修复**:
```tsx
// Before
catch (e) {
  toast.error(editRecordId ? '出库更新失败' : '出库登记失败')
}

// After
catch (e: any) {
  const msg = e?.response?.data?.error?.message || e?.message || (editRecordId ? '出库更新失败' : '出库登记失败')
  toast.error(msg)
}
```

### 7.4 盘点创建弹窗修复

**文件**: `前端代码/src/pages/inventory/components/StocktakingCreateModal.tsx`
**修复**: 添加缺失的 `typeOptions` 和 `scopeOptions` 常量定义
